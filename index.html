About TrailConnect

TrailConnect is a mobile-first hiking platform designed to help users discover trails, join organized hiking events, and connect with outdoor communities.

Built using React Native and Supabase, the app supports event creation, trail browsing, and user profile management.

Whether you're a beginner or experienced climber, TrailConnect helps you plan adventures and explore the Philippines safely.
